## [Brief](#Index)

A small common lisp compiler, written in Rust.

Uses:

 - [Pest](https://crates.io/crates/pest) for parsing and
 - [Cranelift](https://crates.io/crates/cranelift) for codegen.

## [Examples](#Index)

 - `cargo run -- some.lisp`
